Программа для управления централизованной системой учета складских запасов, специально разработанная для мебельных магазинов. 
Она отслеживает поступления и продажи мебели, а также поставщиков, розничные точки и покупателей.

Система позволяет регетрировать события как:
Приход мебели
Продажа мебели,
Возврат мебели.
А так же ведёт учет внутренних данных:
Покупатели,
Поставщики,
Магазины,
Типы продукции.
записи сохраняються в базу данных SQLite и после могут быть просмотренны пользователем.

Программатак же позволяет просматривать все данные через удобнный и интуитивный интерфейс,
а так позволяет формировать и выыводить продвинутые SQL запросы.

Система обращения к базе данных реализована через фасад, что позволяет легко изменить СУБД на другую.
