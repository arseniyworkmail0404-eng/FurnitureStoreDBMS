Программа для управления централизованной системой учета складских запасов, специально разработанная для мебельных магазинов. 
Оно отслеживает поступления и продажи мебели, а также поставщиков, розничные точки и покупателей.

Программа на WIndows. Реализована на языке C# используя библиотеку Windows Forms 

Система позволяет регестрировать события:
  * Приход мебели
  * Продажа мебели
  * Возврат мебели

А так же ведёт учет внутренних данных:
  * Покупатели
  * Поставщики
  * Магазины
  * Типы продукции

Записи сохраняються в базу данных SQLite и после могут быть просмотренны пользователем.

Программа так же позволяет просматривать все данные через удобнный и интуитивный интерфейс,
а так позволяет формировать и выводить продвинутые SQL запросы.

Система обращения к базе данных реализована через фасад, что позволяет легко изменить СУБД на другую.
